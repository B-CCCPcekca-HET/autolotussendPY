ver 1.0 автоматически отправляет почту по расписанию.

settings.txt  ( кодировка UTF-8) содержит настройки
1 строка КОМУ
2 строка КОПИЯ (несколько адресатов идут через запятую, избегайте  лишний  пробелов)
3 строка ТЕМА ПИСЬМА
4 строка ВРЕМЯ ОТПРАВКИ

Заполняем файл один раз, запускаем файл, лотус  не закрываем. Программа сработает толкьо при запущенном  лотусе. В момент запуска  желателдьно не производить никаких работ  на компьютере, так  как будет перехвачено управление  мышкой  и клавой.
